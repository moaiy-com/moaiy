# Security Policy

## Supported Versions

This repository is actively supported on the latest `main` branch and the
latest stable release.

| Version | Supported |
| --- | --- |
| `main` | :white_check_mark: |
| Latest release | :white_check_mark: |
| Older releases | :x: |

## Reporting A Vulnerability

Please do **not** report vulnerabilities in public issues.

Report privately via:

- Email: `security@moaiy.com`
- GitHub private vulnerability reporting (Security Advisories), if available

Please include:

- Affected version / commit
- Reproduction details
- Impact assessment
- Any suggested remediation

## Response Timeline

- Initial acknowledgement: within 72 hours
- Triage update: within 7 business days
- Remediation target: based on severity and exploitability
