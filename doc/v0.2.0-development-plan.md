# v0.2.0 Development Plan - All-in-One Key Management

> **Version**: 0.2.0  
> **Branch**: plan/v0.2.0  
> **Created**: 2026-03-26

## Overview

### Core Design Philosophy
**All-in-One Key Management** - Consolidate all key operations into a single view, simplifying interaction through drag-and-drop and menus.

### Key Changes from v0.1.x

| Before (v0.1.x) | After (v0.2.0) |
|-----------------|----------------|
| Separate Encryption view | Integrated into Key Management |
| Key Detail navigation | All actions via Key Menu (...) |
| Manual operation selection | Auto-detect: encrypted → decrypt, unencrypted → encrypt |
| Multiple specialized views | Single unified Key Management view |

## Feature Details

### 1. Key Management View (All-in-One)

**Key Item Components:**
- Key info display (name, email, type, trust level, expiration)
- **Drop Zone**: Gray dashed rectangle with hint text "Drop files to encrypt/decrypt"
- **Encrypt Button**: Quick action for encryption
- **... Menu**: All other operations

**Key Menu Actions:**
| Action | Description |
|--------|-------------|
| Encrypt | Encrypt files with this key |
| Decrypt | Decrypt files (requires private key) |
| Sign | Sign files or text |
| Verify | Verify signatures |
| Backup | Backup key to file |
| Export Public Key | Export public key to share |
| Share Public Key | Share via system share sheet |
| Upload to Keyserver | Upload to keys.openpgp.org |
| Trust Management | Manage key trust level |
| Edit Key | Edit key properties |
| Delete Key | Remove key from keyring |

### 2. Drag-and-Drop Auto-Detection

**Detection Flow:**
```
┌─────────────────────────────────────────────────────┐
│                    File Dropped                      │
└───────────────────┬─────────────────────────────────┘
                    ▼
┌─────────────────────────────────────────────────────┐
│  Step 1: Quick Binary Signature Detection           │
│  (Read first 100 bytes)                             │
│  • ASCII Armor: "-----BEGIN PGP..."                 │
│  • Binary: Valid OpenPGP Packet Tag                 │
└───────────────────┬─────────────────────────────────┘
                    │
        ┌───────────┴───────────┐
        ▼                       ▼
   Match Found            No Match
        │                       │
        ▼                       ▼
┌───────────────┐       ┌───────────────┐
│ Step 2: GPG   │       │ Not GPG File  │
│ Verification  │       │ → Encrypt     │
└───────┬───────┘       └───────────────┘
        │
    ┌───┴───┐
    ▼       ▼
  Valid   Invalid
    │       │
    ▼       ▼
  Decrypt  Encrypt
```

**File Type Detection (GPGFileTypeDetector):**

| Type | Detection Method | Action |
|------|------------------|--------|
| Encrypted | Packet tags 1, 3, 9, 18 or "BEGIN PGP MESSAGE" | Decrypt |
| Public Key | Packet tags 5, 6, 14 or "BEGIN PGP PUBLIC KEY" | Import |
| Private Key | "BEGIN PGP PRIVATE KEY" | Import |
| Signature | Packet tags 2, 4 or "BEGIN PGP SIGNATURE" | Verify |
| Not GPG | No valid OpenPGP signature | Encrypt |
