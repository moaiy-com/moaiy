## 2. 创建项目

### 2.1 新建文件
1. 点击 Figma 首页的 **"New design file"**
2. 文件会自动命名为 "Untitled"
3. 点击文件名，重命名为 **"Moaiy Design System"**

### 2.2 创建页面结构
在左侧面板的 "Page 1" 上右键，创建以下页面：

```
📁 Moaiy Design System
├── 📄 Foundations（基础样式）
├── 📄 Components（组件库）
├── 📄 Screens（页面设计）
└── 📄 Playground（测试区域）
```

操作步骤：
1. 点击 "Page 1" 旁的 **"+"** 号
2. 双击页面名称重命名
3. 拖动调整顺序

### 2.3 创建 Frame（画框）

切换到 **"Foundations"** 页面：

1. 按 **F** 键选择 Frame 工具
2. 在右侧面板选择 **"Desktop"** → **"MacBook Pro (16-inch)"**（或其他尺寸）
3. 这会创建一个 1920×1080 的画框

或者手动输入尺寸：
1. 在画布上拖动创建任意大小 Frame
2. 在右侧面板输入宽度 `1920`，高度 `3000`（足够放所有基础元素）

## 3. 设置颜色样式

### 3.1 创建颜色色板

**步骤 1：创建颜色块**

1. 按 **R** 键选择矩形工具
2. 在画布上绘制一个 **100×60** 的矩形
3. 在右侧面板 → **Fill**（填充）点击颜色块
4. 输入颜色值，例如：`#4A5568`

**步骤 2：创建颜色样式**

1. 选中矩形
2. 在右侧面板 → **Fill** 区域，点击右侧的 **"四个点"图标** ⋮⋅⋅
3. 选择 **"Create Style"**（创建样式）
4. 命名为：`Primary/Moai Stone`
5. 点击 **"Create Style"**

**步骤 3：批量创建颜色**

重复上述步骤，创建所有颜色：

```

├── Moai Stone (#4A5568)
├── Moai Dark (#2D3748)
└── Moai Light (#718096)

Accent/
├── Security Green (#48BB78)
├── Security Green Dark (#38A169)
└── Security Green Light (#9AE6B4)

Semantic/
├── Success (#48BB78)
├── Warning (#F6AD55)
├── Error (#FC8181)
└── Info (#4299E1)

Neutral/Light/
├── Background (#F7FAFC)
├── Surface (#FFFFFF)
├── Border (#E2E8F0)
├── Text Primary (#1A202C)
├── Text Secondary (#4A5568)
└── Text Tertiary (#718096)

Neutral/Dark/
├── Background (#1A202C)
├── Surface (#2D3748)
├── Border (#4A5568)
├── Text Primary (#F7FAFC)
├── Text Secondary (#E2E8F0)
```

### 3.2 颜色样式命名规范

使用 **"/"** 分层级，例如：
- `Primary/Moai Stone`
- `Semantic/Success`
- `Neutral/Light/Background`

这样在右侧面板的样式列表中会自动分组显示。

## 4. 设置文字样式

### 4.1 创建文字样式

**步骤 1：创建文字**

1. 按 **T** 键选择文字工具
2. 在画布上点击并输入示例文字，如 "Display"
3. 在右侧面板设置字体属性

**步骤 2：设置字体属性**

- **Font**: SF Pro（如果没有，用 Inter 或 system-ui 代替）

- **Weight**: Semibold (600)

## 5. 创建基础组件

切换到 **"Components"** 页面

### 5.1 创建按钮组件

**步骤 1：绘制按钮**

1. 按 **R** 键绘制一个矩形（宽度自动，高度 36）
2. 设置圆角：在右侧面板 → **"Corner radius"** 输入 **8**
3. 填充颜色：使用颜色样式 `Accent/Security Green`

**步骤 2：添加文字**

1. 按 **T** 键，在按钮上点击
2. 输入 "Button"
3. 设置文字样式为 `Headline`
4. 文字颜色设为白色 `#FFFFFF`
5. 居中对齐

**步骤 3：添加内边距**

1. 同时选中矩形和文字
2. 在右侧面板 → **"Auto Layout"** 点击 **"+"**
3. 设置：
   - Padding: **上下 10，左右 20**
   - Direction: **Horizontal**
   - Align: **Center**

**步骤 4：创建组件**

1. 选中整个按钮（Auto Layout 组）
2. 按 **⌥ + ⌘ + K**（Mac）或 **Ctrl + Alt + K**（Windows）
3. 组件创建成功，左侧面板会显示菱形图标 ◇

**步骤 5：创建变体（Variants）**

这是最重要的部分！我们需要创建按钮的不同状态：

1. 选中组件，按 **⌘ + D** 复制 3 份
2. 修改每个副本的颜色：

| 变体名称 | 背景色 | 文字色 | 边框 |
|---------|--------|--------|------|
| Primary | Security Green | White | 无 |
| Secondary | 透明 | Security Green | 1px Security Green |
| Tertiary | 透明 | Text Primary | 无 |
| Destructive | Error | White | 无 |

3. 选中所有 4 个按钮
4. 在右侧面板 → **"Properties"** 区域，点击 **"Add Property"** → **"Variant"**
5. Figma 会自动将它们合并为一个组件集

**步骤 6：添加状态变体**

每个按钮还需要 4 种状态：Default、Hover、Pressed、Disabled

1. 在现有变体基础上，再复制一份
2. 修改颜色：

以 Primary 为例：
- **Default**: 背景 #48BB78
- **Hover**: 背景 #38A169
- **Pressed**: 背景 #2F855A
- **Disabled**: 背景 #9AE6B4，文字 50% 透明度

最终组件结构：

```
◇ Button
├── Primary/Default
├── Primary/Hover
├── Primary/Pressed
├── Primary/Disabled
├── Secondary/Default
├── Secondary/Hover
├── Secondary/Pressed
├── Secondary/Disabled
├── ...（其他类型）
└── Destructive/Disabled
```

### 5.2 创建输入框组件

**步骤 1：绘制输入框**

1. 绘制矩形：宽 300，高 36
2. 圆角：8
3. 填充：使用样式 `Neutral/Light/Surface`
4. 描边：1px，使用样式 `Neutral/Light/Border`

**步骤 2：添加占位文字**

1. 添加文字："请输入..."
2. 样式：`Body`
3. 颜色：`Neutral/Light/Text Tertiary`

**步骤 3：创建 Auto Layout**

1. 选中矩形和文字
2. 添加 Auto Layout
3. 左右 Padding：14

**步骤 4：创建变体**

| 变体 | 边框颜色 |
|------|----------|
| Default | Border |
| Focus | Security Green（2px） |
| Error | Error（1px） |
| Disabled | Border（背景灰色） |

### 5.3 创建卡片组件

**步骤 1：绘制卡片**

1. 绘制矩形：宽 400，高自动
2. 圆角：12
3. 填充：Surface
4. 添加阴影：
   - 点击 Fill 下方的 **"Effects"**
   - 点击 **"+"** → **"Drop Shadow"**
   - 设置：X=0, Y=1, Blur=3, Color=rgba(0,0,0,0.08)

**步骤 2：添加内容**

```
┌─────────────────────────────────────┐
│  🔐 主要密钥                    ⋯  │  ← 标题行
│  ─────────────────────────────────  │  ← 分割线
│  👤 Alice (alice@example.com)       │  ← 内容
│  📅 2024年1月1日 · RSA-4096         │
│  ─────────────────────────────────  │  ← 分割线
│  [🔒 加密] [📤 分享] [💾 备份]      │  ← 操作按钮
└─────────────────────────────────────┘
```

**步骤 3：使用 Auto Layout 组织**

1. 创建标题行（图标 + 文字 + 更多按钮）
2. 创建内容区
3. 创建操作区（多个按钮）
4. 将三部分用 Auto Layout 组合，间距 16

## 6. 创建复合组件

### 6.1 密钥卡片（Key Card）

这是一个特殊的卡片组件，专门用于显示密钥信息：

**结构：**
```
◇ Key Card
├── Header（头部）
│   ├── Icon（图标）
│   ├── Title（标题）
│   └── More Button（更多按钮）
├── Divider（分割线）
├── Content（内容）
│   ├── Name + Email
│   └── Date + Algorithm
├── Divider（分割线）
├── Status（状态）
│   └── Badge（徽章）
├── Divider（分割线）
└── Actions（操作）
    ├── Encrypt Button
    ├── Share Button
    └── Backup Button
```

**创建步骤：**

1. **创建各个子元素**
2. **用 Auto Layout 组装**
3. **创建组件**
4. **添加变体**（有效、即将过期、已过期）

### 6.2 Drop Zone（拖拽区域）

**结构：**
```
◇ Drop Zone
├── Icon（📄 图标）
├── Primary Text（主标题）
├── Secondary Text（副标题）
└── Supported Types（支持的文件类型）
```

**样式：**
- 边框：2px 虚线
- 背景：透明或浅色
- 最小高度：200px

**变体：**
- Default（默认）
- Hover（拖拽悬停时，边框变绿色）

### 6.3 状态徽章（Status Badge）

**结构：**
```
◇ Status Badge
├── Icon（✅ ⚠️ ❌）
└── Text（状态文字）
```

**样式：**
- 圆角：Full（9999px，即胶囊形）
- 内边距：4px 10px
- 字体：Caption 1 Medium

**变体：**
| 状态 | 背景 | 文字 | 图标 |
|------|------|------|------|
| Success | #9AE6B4 | #276749 | ✅ |
| Warning | #FEEBC8 | #C05621 | ⚠️ |
| Error | #FED7D7 | #C53030 | ❌ |
| Info | #BEE3F8 | #2B6CB0 | ℹ️ |

## 7. 创建页面设计

切换到 **"Screens"** 页面

### 7.1 创建主界面 Frame

1. 按 **F** 键
2. 选择 **"Desktop"** → **"MacBook Pro (16-inch)"** 或自定义尺寸
3. Frame 名称：`Main Window`

### 7.2 设计主界面布局

**布局结构：**
```
┌─────────────────────────────────────────────────────────────┐
│  [Toolbar Area]                                             │
├──────────────┬──────────────────────────────────────────────┤
│              │                                              │
│   Sidebar    │              Main Content                    │
│   (220px)    │                                              │
│              │                                              │
│   🔑 密钥    │   ┌────────────────────────────────────┐    │
│   🔒 加密    │   │                                    │    │
│   📋 历史    │   │         Key List                   │    │
│   ⚙️ 设置    │   │         (Cards)                    │    │
│              │   │                                    │    │
│              │   └────────────────────────────────────┘    │
│              │                                              │
└──────────────┴──────────────────────────────────────────────┘
```

**创建步骤：**

1. **创建 Toolbar**
   - 高度：52px
   - 背景：Surface
   - 底部边框：1px Border

2. **创建 Sidebar**
   - 宽度：220px
   - 背景：Surface 或稍深的颜色
   - 使用 Auto Layout 排列导航项

3. **创建主内容区**
   - 背景：Background
   - 内边距：20px
   - 放置密钥卡片列表

### 7.3 使用组件

**从组件库拖入组件：**

1. 打开左侧面板 → **"Assets"** 标签
2. 搜索或浏览你创建的组件
3. 将组件拖入画布

**修改组件实例：**

1. 拖入组件后，它是一个"实例"
2. 你可以修改实例的文字内容
3. 但样式（颜色、字体）会保持与主组件同步

**切换变体：**

1. 选中组件实例
2. 在右侧面板 → **"Properties"** 区域
3. 通过下拉菜单切换不同变体

### 7.4 创建响应式布局

使用 **Constraints**（约束）让布局适应不同尺寸：

1. 选中元素
2. 在右侧面板 → **"Constraints"** 区域
3. 设置约束方式：
   - Sidebar：左、上、下固定，宽度固定
   - Main Content：四边固定
   - 卡片：左右固定，上固定

## 8. 使用技巧

### 8.1 快速复制样式

**从一个元素复制样式到另一个：**

1. 选中源元素
2. 按 **⌥ + ⌘ + C**（复制样式）
3. 选中目标元素
4. 按 **⌥ + ⌘ + V**（粘贴样式）

### 8.2 批量修改颜色

**如果你想全局修改某个颜色：**

1. 在右侧面板 → **"Selection"** 标签旁点击 **"Library styles"** 图标
2. 找到要修改的颜色样式
3. 点击编辑图标
4. 修改颜色值
5. 所有使用该颜色的元素都会自动更新

### 8.3 使用插件

**推荐插件：**

1. **Unsplash** - 免费高质量图片
   - 安装：Plugins → Browse plugins → 搜索 "Unsplash"

2. **Iconify** - 海量图标
   - 安装：搜索 "Iconify"
   - 使用：可以直接搜索 SF Symbols

3. **Content Reel** - 快速填充示例内容
   - 安装：搜索 "Content Reel"

**安装插件方法：**
1. 点击右上角 **"Resources"** 图标（书本形状）
2. 点击 **"Plugins"** 标签
3. 搜索并安装

### 8.4 创建交互原型

**添加交互：**

1. 选中元素
2. 在右侧面板切换到 **"Prototype"** 标签
3. 点击元素右侧的 **"+"** 号
4. 拖动连线到目标 Frame
5. 设置触发方式（点击、悬停等）
6. 设置动画效果

**预览原型：**

1. 点击右上角 **"Play"** 按钮
2. 在新窗口中预览交互效果

### 8.5 导出设计

**导出图片：**

1. 选中要导出的 Frame 或元素
2. 在右侧面板 → **"Export"** 区域
3. 点击 **"+"** 添加导出设置
4. 选择格式（PNG、JPG、SVG、PDF）

6. 点击 **"Export"** 按钮

**导出 CSS 代码：**

1. 选中元素
2. 切换到右侧 **"Inspect"** 标签
3. 查看自动生成的 CSS 代码
4. 可以复制使用

## 10. 常见问题

### Q1: 如何让组件自适应内容高度？

**A:** 使用 Auto Layout
1. 选中组件内容
2. 添加 Auto Layout
3. 设置为垂直方向
4. 高度会自动适应内容

### Q2: 如何创建响应式设计？

**A:** 结合 Auto Layout 和 Constraints
- Auto Layout 控制内部元素的排列
- Constraints 控制整体相对于父容器的位置

### Q3: 如何分享设计给开发者？

**A:** 三种方式
1. **分享链接**：点击右上角 "Share" → 生成链接
2. **开发者模式**：开发者可以在 "Inspect" 标签查看代码
3. **导出标注**：使用插件导出详细标注

### Q4: 如何保持设计一致性？

**A:** 
1. 始终使用样式（颜色、文字）
2. 始终使用组件
3. 定期整理和更新设计系统
