# v0.5.0 Minimal Release Checklist (AI-Accelerated)

## Scope

This checklist is optimized for an early-stage open-source app where most implementation work is AI-assisted ("vibe coding"), while keeping critical quality gates intact.

- Target release date: 2026-04-10
- Release branch: `release/v0.5.0`
- Freeze rule: bug fixes only after D1

## Go/No-Go Gates

All gates must pass before publishing:

- `P0 = 0`
- `P1 = 0`
- Core end-to-end flows pass (manual + automated)
- No high-risk security findings remain open
- English + Simplified Chinese UI strings render correctly in core screens
