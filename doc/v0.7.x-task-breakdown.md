# Moaiy v0.7.x Task Breakdown

## Scope Summary

- Target upgrade path: `0.6.5 -> 0.7.x`
- Primary epic: subkey and trust lifecycle hardening
- v0.7.0 priority: revocation center, ownertrust transfer, and key-signing menu gate

## v0.7.0 (P0)

1. Add revocation center flow:
- Generate ASCII-armored revocation certificate for a key.
- Import revocation certificate into local keyring.

2. Add ownertrust transfer flow:
- Export ownertrust records for migration/backup.
- Import ownertrust records and refresh trustdb.

3. Expose `Sign Key` menu via user-configurable flag:
- Keep safe default (`off`).
- Add setting switch for explicit enablement.

4. UI entry points:
- Add `action_ownertrust_transfer` and `action_manage_revocation` in key action menu.
- Add `OwnerTrustTransferSheet` and `RevocationCenterSheet`.

5. Localization:
- Add new keys for v0.7.0 flows in `Localizable.xcstrings` with full required locales.

6. Validation:
- Unit tests for menu availability and new flow gating.
- Localization integrity check and CI build/test pass.

## v0.7.1 (P1)

1. Expand subkey lifecycle:
- Revoke subkey
- Disable/enable subkey
- Add rotation guidance

2. Improve subkey state UX:
- Add high-risk/expiring status indicators.
- Improve filtering and sort controls.

## v0.7.2 (P1/P2)

1. Smart-card advanced operations:
- PIN/Admin PIN management
- Key-to-card guided flow
- Enhanced diagnostics

2. Error boundary hardening:
- Better fallback and remediation hints for card daemon/device errors.

## v0.7.3 (P2)

1. Policy profile support:
- Cipher/digest policy presets.
- Team-friendly security profiles.

2. Sync and operational tooling:
- Key refresh workflows (keyserver/WKD paths)
- Sanitized audit export

## Release Gate Checklist (for every v0.7.x release)

1. `./scripts/check_localization.py --catalog Moaiy/Resources/Localizable.xcstrings`
2. `xcodebuild build` and `xcodebuild test` must pass in CI
3. Manual smoke:
- Create key
- Import key
- Sign/verify
- Subkey flow
- New trust/revocation flows
4. Update:
- `CHANGELOG.md`
- docs for newly exposed user-facing capabilities
