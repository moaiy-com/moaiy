# v0.5.0 External Validation Pack

## Purpose

Close remaining release gates that cannot be completed in the current local environment:

1. Real Intel machine smoke validation.
2. macOS 12.x smoke validation.
3. Remaining D4 manual UI click-through records (`MR-01` to `MR-05`, `MR-08`, `MR-11`).

## One-Command Smoke Validation

Use script:

`./scripts/v050_external_validation.sh`

The script runs the v0.5.0 target smoke bundle and exports:

- `xcresult` bundle
- JSON summary
- Markdown summary snippet

## Run On Required Environments

### A) Real Intel machine

```bash
cd /path/to/moaiy
./scripts/v050_external_validation.sh --label intel-real
```

If no Intel machine is available pre-release, keep this item open and use temporary fallback evidence:

```bash
cd /path/to/moaiy
./scripts/v050_external_validation.sh --label intel-fallback --arch x86_64
```

### B) macOS 12.x machine

```bash
cd /path/to/moaiy
./scripts/v050_external_validation.sh --label macos12-real
```

## Expected Pass Criteria

- Script exit code: `0`
- Summary result: `Passed`
- `failedTests = 0`
- Current baseline target: `93/93`

If pass count differs from `93`, treat as warning and verify that the same test target set was executed.

## Output Locations

Default output directory:

`dist/validation`

Per run, generated files:

- `v0.5.0-<label>-<timestamp>.xcresult`
- `v0.5.0-<label>-<timestamp>-summary.json`
- `v0.5.0-<label>-<timestamp>-summary.md`
- `v0.5.0-<label>-<timestamp>-xcodebuild.log`

When known blocker `RB-002` is detected, script also generates:

- `v0.5.0-<label>-<timestamp>-fallback-command.sh`

## Troubleshooting (RB-002)

If script exits with code `3` and summary shows `Blocked` with `RB-002`:

- Cause: sandboxed environment failed Swift macro expansion (`ObservationMacros` / `PreviewsMacros`).
- This is environment-specific and does not represent product-level regression by itself.

Use generated fallback script on target machine outside restricted sandbox:

```bash
cd /path/to/moaiy
bash dist/validation/v0.5.0-<label>-<timestamp>-fallback-command.sh
```

Then re-run `xcrun xcresulttool` summary extraction if needed, or just re-run:

```bash
cd /path/to/moaiy
./scripts/v050_external_validation.sh --label <label>
```
