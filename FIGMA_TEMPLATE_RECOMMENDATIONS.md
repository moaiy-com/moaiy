## 🚀 推荐使用流程

### 方案 A：使用 Apple Design Resources（推荐）

**优势**:
- ✅ 最符合 Moaiy 的 macOS 原生风格
- ✅ 完整的组件库
- ✅ 官方维护

**步骤**:
1. 下载 Apple Design Resources - macOS
2. 创建新的 Figma 文件
3. 从模板中选择 macOS 窗口框架
4. 设计 Moaiy 的核心界面

**预计时间**: 1-2 小时完成核心界面设计

---

### 方案 B：混合使用（更灵活）

**组合**:
1. **主框架**: Apple Design Resources - macOS
2. **数据展示**: Free Admin Dashboard UI Kit
3. **应用图标**: macOS App Icon Template

**优势**: 兼顾原生外观和丰富的数据展示组件

## 💡 Moaiy 核心界面设计建议

基于模板，你需要设计以下界面：

### 1. 主界面（密钥管理）
- **布局**: 侧边栏 + 主内容区
- **组件**:
  - 侧边栏导航（密钥、加密、设置）
  - 密钥列表（表格或卡片视图）
  - 操作按钮（添加、导入、导出）

### 2. 密钥详情页
- **布局**: 详情视图
- **组件**:
  - 密钥信息卡片
  - 操作按钮（加密、导出、删除）
  - 使用历史

### 3. 加密界面
- **布局**: 标签页（文本/文件）
- **组件**:
  - 文本输入区
  - 文件拖放区
  - 收件人选择器
  - 加密按钮

### 4. 设置界面
- **布局**: 分组设置
- **组件**:
  - 设置分组
  - 开关和输入框
  - 保存按钮
