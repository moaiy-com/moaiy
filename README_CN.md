# Moaiy

> **像摩艾一样守护您的秘密**

**[English Version](./README.md)**

## 项目概述

**Moaiy** 是一个傻瓜化、图形化的GPG密钥管理工具，源自摩艾石像 (Moai)，象征神秘感、安全性、长久性。

让信息加密走近普通用户，不再是极客们的专属工具。

## 核心特色

🗿 **神秘感** - 守护您的数字秘密，让敏感信息永远只属于您自己

🔐 **安全性** - 提供军事级别的加密保护，让您的数据固若金汤

⏳ **长久性** - 设计理念是长期可靠，一次加密，永久保护

## 目标用户

- **加密货币用户**: 需要安全存储助记词、私钥
- **隐私保护者**: 希望加密个人文件和通信
- **普通办公用户**: 需要加密敏感文档

## 技术栈

- **开发语言**: Swift 6.2
- **UI框架**: SwiftUI
- **最低支持**: macOS 12.0+
- **加密工具**: 内置 GPG (完全自包含)

## 功能亮点

### 零门槛使用
- 双击即用，无需任何技术背景
- 完全自包含，无外部依赖
- 5分钟完成首次加密

### 智能化设计
- 自动配置和智能推荐
- 智能错误诊断和自动修复

- 完全符合 App Store 审核要求
- 硬件密钥支持（Pro版）
